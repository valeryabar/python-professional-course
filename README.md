# My Python Learning

В этом репозитории собраны мои некоторые решения инженерных задач на Python. Я самостоятельно изучаю программирование, фокусируясь на эффективной обработке данных и автоматизации.

## Что здесь есть:
*   **zip_tree_viewer.py** — Системная утилита для визуализации структуры ZIP-архивов с конвертацией размеров файлов в человекочитаемый вид (KB/MB/GB).
*   **revenue_calculator.py** — Инструмент для агрегации финансовых данных из нескольких CSV-отчетов и сопоставления их с прайс-листом в формате JSON.
*   **name_log_parser.py** — Оптимизированный скрипт для частотного анализа логов с использованием `collections.Counter`.
*   **hangman.py** — Консольная игра с реализованной логикой валидации ввода и визуализацией состояния.

## Технологии и навыки:
- Работа со стандартной библиотекой Python (os, sys, json, csv, zipfile, collections).
- Понимание алгоритмической сложности и оптимизация по памяти.
- Опыт самостоятельного поиска решений и работы с документацией.
